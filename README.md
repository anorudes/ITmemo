# ITmemo (working title)
![](http://anorudes.github.io/ITmemo/img.png)

## Install
```
npm install
jspm install
cd client
bower install
```

## Start (with server)
```
gulp
http://localhost:3000/
```

## Start (nw.js)
```
gulp --nw
```
