# inSRC

## Description
![](http://anorudes.github.io/inSRC/img.png)

## Install
```
npm install
jspm install
cd client
bower install
```

## Start (with server)
```
gulp
http://localhost:3000/
```

## Start (nw.js)
```
gulp --nw
```

## Buld app (windows)
```
gulp build
```
